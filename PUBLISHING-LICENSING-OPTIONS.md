#### CC BY-SA (Attribution-ShareAlike)
**"Share and adapt, but derivatives must use same license"**

**Best for:** Community knowledge, collaborative works

**Allows:**
- âœ“ Copy and redistribute

#### CC BY-NC (Attribution-NonCommercial)
**"Share and adapt, but not for commercial use"**

**Best for:** Music, art, personal creative works

**Allows:**

#### CC BY-NC-SA (Attribution-NonCommercial-ShareAlike)
**"Share and adapt non-commercially, derivatives must match"**

**Best for:** Community resources, open-source content

**Allows:**
- âœ“ Copy and redistribute (non-commercial)
- âœ“ Remix, transform, build upon (non-commercial)

**Requires:**
- Must give appropriate credit
- Cannot use commercially
- Derivatives must use same license

---

#### CC BY-ND (Attribution-NoDerivatives)
**"Share as-is only, no modifications"**

**Best for:** Poetry, finished artwork, sacred texts

**Allows:**
- âœ“ Copy and redistribute (even commercially)

**Requires:**
- Must give appropriate credit
- **Cannot modify, transform, or build upon**

**Display:**
```
Licensed under Creative Commons Attribution-NoDerivatives 4.0 (CC BY-ND 4.0)

You may copy and redistribute in any medium or format for any purpose,
even commercially.

You must give credit and cannot modify the work.
```

---

#### CC BY-NC-ND (Attribution-NonCommercial-NoDerivatives)
**"Share as-is only, non-commercial"**

**Most restrictive Creative Commons license**

**Best for:** Personal writing, intimate poetry, sacred music

**Allows:**
- âœ“ Copy and redistribute (non-commercial only)

**Requires:**
- Must give appropriate credit
- Cannot modify
- Cannot use commercially

### 3. Public Domain Dedications

#### CC0 (No Rights Reserved)
**"Give it to the world, no restrictions"**

**Best for:** Freely sharing knowledge, community resources

**Allows:**
- âœ“ Copy, modify, distribute, perform
- âœ“ Commercial use
- âœ“ No attribution required

**Display:**
```
CC0 1.0 Universal - Public Domain Dedication

To the extent possible under law, [Creator] has waived all copyright
and related rights to this work. You can copy, modify, distribute and
perform the work, even for commercial purposes, all without asking permission.
```

### 4. Custom Licenses

Creators can write their own terms:

**Examples:**
- "Free for personal use, contact for commercial licensing"
- "Free for Soullab members, paid for others"
- "Educational use permitted, commercial requires license"
- "Attribution required for spiritual/healing practitioners"

**Platform Support:**
- Custom license text field
- License file upload (.pdf, .txt)
- License URL for detailed terms

### License Types Enum

```typescript
type LicenseType =
  | 'all-rights-reserved'      // Full copyright
  | 'cc-by'                     // CC Attribution
  | 'cc-by-sa'                  // CC Attribution-ShareAlike
  | 'cc-by-nc'                  // CC Attribution-NonCommercial
  | 'cc-by-nc-sa'               // CC BY-NC-SA
  | 'cc-by-nd'                  // CC Attribution-NoDerivatives
  | 'cc-by-nc-nd'               // CC BY-NC-ND
  | 'cc0'                       // Public Domain
  | 'custom';                   // Custom license

interface LicenseInfo {
  type: LicenseType;
  name: string;
  shortName: string;
  url?: string;
  commercial_use: boolean;
  derivatives: boolean;
  attribution_required: boolean;
  share_alike?: boolean;
  custom_text?: string;
  custom_file_url?: string;
}
```
