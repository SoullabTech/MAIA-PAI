   - **Issue:** "ELEMENTAL" and "ALCHEMY" compete for attention
   - **Impact:** Eye doesn't know where to land first
   - **Reader response:** Slightly confusing

   - **Issue:** No subtitle visible

   - **Issue:** Not visible in image provided

   - **Must include:** "Kelly Nezat"

   - **Issue:** Hard to read where elements are marked
   - **Impact:** Readers may not understand the elemental framework
   - **Enhancement:** Make more prominent or use icons

5. **Contrast & Readability**
   - **Issue:** Title text blends into busy background
   - **Impact:** May not "pop" in thumbnail (critical for online sales)
   - **Enhancement:** Needs better contrast/glow/shadow

6. **Sacred Geometry Definition**
   - **Issue:** Grid is present but faint
   - **Impact:** Key visual metaphor (torus of change) not clear
   - **Enhancement:** Strengthen key geometric lines

## TECHNICAL SPECIFICATIONS

### Final Cover Dimensions:
- **Print:** 6" x 9" (standard trade paperback)
  - Bleed: 0.125" on all sides
  - Safe zone: 0.25" from trim edge
  - Spine width: Depends on page count (calculate at printing)

- **E-book:** 2560 x 1600 pixels (Amazon KDP recommended)
  - PNG or JPG format
  - RGB color space
  - 72 DPI minimum (300 DPI preferred)

### Typography Requirements:
- **Title:** Must be readable at 100px wide (thumbnail test)
- **Author:** Minimum 18pt for print
- **Subtitle:** Minimum 12pt for print

### Color Specifications:
- **Print:** CMYK color space
- **Digital:** RGB color space
- Provide both versions if doing print + ebook

### File Deliverables Needed:
1. Full cover (front + spine + back) - print version
2. Front cover only - ebook version
3. High-res versions (300 DPI)
4. Web-ready versions (72 DPI, optimized file size)
5. Source files (layered PSD/AI for future edits)
