# Demon Detection: Language Pattern Recognition

**Purpose**: Recognize shadow patterns through language, NOT to manipulate, but to **see and reflect back** what the reader may not yet see in themselves.

**Ethics**: All detection is transparent. Readers can ask "What are you noticing?" and receive honest answers.

---

## Core Principle

**We're not profiling. We're listening.**

The system tracks language patterns to:
1. Recognize developmental phase
2. Detect shadow material
3. Assess readiness for movement
4. **Show readers their own patterns** when asked

## Detection Algorithm

```typescript
interface DemonDetection {
  demon: string;           // e.g., 'fire_1_refusal'
  confidence: number;      // 0-1
  evidence: string[];      // Actual quotes/keywords found
  phase_suggestion: string; // e.g., 'Fire 1'
  supportive_prompt: string; // What to offer
}

function detectDemons(userText: string, conversationHistory: string[]): DemonDetection[] {
  const detections: DemonDetection[] = [];

  // For each demon pattern
  for (const pattern of demonPatterns) {
    const matches = findKeywords(userText, pattern.keywords);

    if (matches.length >= pattern.threshold) {
      detections.push({
        demon: pattern.name,
        confidence: matches.length / pattern.keywords.length,
        evidence: matches,
        phase_suggestion: pattern.phase,
        supportive_prompt: pattern.reflection_prompt
      });
    }
  }

  // Check conversation history for persistent patterns
  const historicalMatches = analyzeHistory(conversationHistory);

  return detections.sort((a, b) => b.confidence - a.confidence);
}
```

> - "Hell no to this call"
>
> **What this might indicate**:
> - You may be in **Fire 1** (Vision & Refusal)
> - I'm sensing what we call **Refusal of the Call** - not as failure, but as a legitimate pause or a protective response to past wounds
>
> **Why I think this**:
> - Your language has that "I'm tired of being asked for more" quality
> - There's a sense of having given a lot and needing rest
> - Or possibly, past experiences where saying yes led to pain
>
> **What this tells me about what you might need**:
> - Permission to refuse if that's true
> - Space to discern: Is this "not yet" or "not this way" or "I need rest first"?
> - Practices that don't push, but hold
>
> **Does this resonate? Or am I missing something?**
