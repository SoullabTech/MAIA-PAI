- Add headings
- Organize sections
- Create outline

**3. Refinement Mode**
- Turn on MAIA Writing Agent
- Polish each paragraph
- Check consistency
- Deepen flow

**4. Editorial Mode**
- Turn on MAIA Editorial Agent
- Fix grammar
- Improve clarity
- Verify facts

**5. Design Mode**
- Turn on MAIA Design Agent
- Optimize layout
