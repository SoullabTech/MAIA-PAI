# Soullab Academy Video Scripts

Complete video lesson scripts for recording the living transmission of Soullab's vision.

## Overview

These scripts are designed for recording in **Descript** - each one includes:
- Time-stamped sections
- Detailed talking points
- Key phrases to emphasize
- Filming notes (tone, pace, energy)
- Visual suggestions
- Reflection prompts to discuss on camera
- Links to resources

## Recording Workflow

1. **Import script into Descript** as your baseline
2. **Record while reading** (Descript follows along)
3. **Edit naturally** - remove ums, pauses, or re-record sections
4. **Add visuals** where suggested in the script
5. **Export to video** when complete
6. **Upload** and add video URL to lesson in `/lib/academy-curriculum.ts`

## Filming Best Practices

### General Guidelines

- **Tone**: Authentic, grounded, intimate yet powerful
- **Environment**: Clean background, good lighting, minimal distractions
- **Camera**: Eye-level, slightly closer than you think (intimate teaching)
- **Energy**: Match the script notes - build from contemplative to inspired
- **Pauses**: Don't fear silence - let insights land

### Script Usage

- Use the script as **your foundation**, not a rigid teleprompter
- Feel free to **improvise** if inspiration strikes
- The **key phrases** are most important to hit clearly
- **Reflection prompts** should be delivered slowly and with invitation
- **Filming notes** at the end of each script guide tone and pacing

### Visual Elements

Each script suggests visual aids like:
- Diagrams (torus, trinity, elements)
- Sacred geometry animations
- Nature examples
- Book page references

Gather these before filming or add in post-production.
