## How to Use

### When a New Partner Submits Reflection

1. **Automatically (if automation set up):**
   - Note created from template in `Partners/YYYY-Q#/`
   - Prefilled with their elemental responses
   - Slack notification sent

2. **Manually (if not automated yet):**
   - Copy `_Templates/Partner.md`
   - Create new note: `Partners/2025-Q1/FirstName LastName.md`
   - Fill in their responses from Supabase

### Before Meeting

1. Open their note
2. Read twice: once for content, once for feeling
3. Add **EO Notes** section with intuition
4. Prepare 2-3 questions

### After Meeting

1. Capture essence (not transcript)
2. Update **EO Notes** with insights
3. Mark **Status** and **Phase**
4. Note next smallest true step

### Monthly

1. Create reflection note from template
2. Review all active partners
3. Notice patterns across the field
4. Adjust process if needed
