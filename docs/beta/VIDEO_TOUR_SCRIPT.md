## Production Notes

### Screen Recording Tips
- Use **CleanShot X** or **Loom** for clean screen capture
- Record in **1080p or 1440p** (not 4K—file size too large)
- Use **browser fullscreen mode** (hide bookmarks bar, extensions)
- **Slow down** — Give viewers time to see what you're showing

### Audio Tips
- Use a **good microphone** (Blue Yeti, Rode NT-USB, or similar)
- Record in a **quiet space** (no background noise)
- Speak **slowly and calmly** — This isn't a tutorial. It's companionship.
- Leave **pauses** — Let moments breathe

### Editing Tips
- Add **subtle background music** (ambient, not distracting)
- Use **fade transitions** between sections (not cuts)
- Add **text overlays** for key concepts:
  - "Soul Signature"
  - "Anamnesis: Unforgetting"
  - "Morphic Resonance"
- Keep **pacing gentle** — No rush

### Upload
- **Platform:** Loom or YouTube (unlisted)
- **Title:** "Your First Week with MAIA — Beta Tester Guide"
- **Description:** Include links to handbook, feedback email, platform access
- **Thumbnail:** Holoflower with text "Your First Week with MAIA"
