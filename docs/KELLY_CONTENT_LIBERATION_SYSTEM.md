### Phase 2: SORT (The Quick Scan)
**Goal**: Rough categories only - not perfect organization

**Method**: 3-breath decision rule

For each video, take 3 breaths and ask:
- "What is this?" (meditation, journey, teaching, client session, random idea?)
- That's it. One label. Move on.

**Categories** (simple):
1. **Spiralogic Teachings** (the 12 phases + framework explanations)
2. **Meditations** (short daily practices - 5-15 min)
3. **Journey Work** (deep guided journeys - 15-30 min)
4. **Client Sessions** (recordings of actual sessions)
5. **Random Gold** (ideas, fragments, things to mine later)
6. **Archive** (old versions, doesn't fit anywhere, maybe delete)

You tell me which bucket. I do the organizing.

### Phase 3: PROCESS (The Assembly Line)
**Goal**: Transform raw recordings into polished deliverables

**Method**: I handle the production pipeline

**For each video you want to liberate:**

**Step 1: You give me access**
- Share link or file path
- Tell me rough category (meditation? journey? teaching?)
- Tell me if there's anything specific you want extracted

**Step 2: I process it**
- Watch/listen
- Extract transcript
- Identify key teaching/structure
- Note your language patterns, pacing, energy
- Create polished script (edited for clarity, keeping your voice)
- Suggest audio edits if needed (cut tangents, add music cues)
- Format into multiple deliverables:
  - Audio meditation file (for platform)
  - Written script (PDF download)
  - Teaching summary (one-page essence)
  - Integration practices (what to do after)

**Step 3: You review & approve**
- I show you the polished version
- You say: "Yes, ship it" or "Tweak this part"
- Takes you 5-10 minutes instead of 5-10 hours

**Step 4: I integrate into platform**
- Add to your content library
- Tag appropriately (element, phase, use case)
- Create pathways (if someone does X, suggest Y next)
