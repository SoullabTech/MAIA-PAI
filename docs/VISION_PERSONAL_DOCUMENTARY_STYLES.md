## Implementation Architecture

### Database Schema

```typescript
interface MemberDocumentaryStyle {
  memberId: string;
  selectedPreset: 'sandworm' | 'cosmos' | 'manuscript' | 'forest' | 'ocean' | 'urban';
  customizations?: {
    // Allow minor tweaks within preset constraints
    particleDensity?: 'sparse' | 'medium' | 'dense';
    motionSpeed?: 'slow' | 'medium' | 'fast';
    contrastLevel?: 'low' | 'medium' | 'high';
  };
  appliedToPages: ('story' | 'astrology' | 'main' | 'upper')[];
  createdAt: Date;
  updatedAt: Date;
}
```

### Component Structure

```typescript
// Wrapper that applies documentary style
<DocumentaryStyleProvider preset={member.documentaryStyle}>
  {/* All pages inherit atmospheric layers */}
  <StoryPage />
</DocumentaryStyleProvider>

// Each preset provides:
interface DocumentaryPreset {
  name: string;
  description: string;
  previewImage: string;

  colors: {
    background: string[];  // Gradient stops
    text: string[];
    accent: string[];
    glow: string;
  };

  particles: {
    type: 'stars' | 'sand' | 'fireflies' | 'bubbles' | 'rain' | 'lights';
    count: number;
    color: string;
    speed: number;
  }[];

  atmosphere: {
    haze?: RadialGradient[];
    texture?: 'film-grain' | 'paper' | 'canvas' | 'concrete';
    vignette?: boolean;
    depth?: 'shallow' | 'medium' | 'deep';
  };

  typography: {
    fontFamily: string;
    letterSpacing: string;
    textShadow: string;
  };

  motion: {
    breathingSpeed: number;
    pulseIntensity: number;
    driftDirection: 'up' | 'down' | 'horizontal';
  };
}
```

## Aesthetic Coherence Strategy

### How We Prevent "Menagerie of Clashing Aesthetics"

**Rule 1: Curated Presets Only**
- No "build your own theme" (that's where chaos lives)
- Only designer-curated documentary styles
- Each preset professionally balanced

**Rule 2: Soullab Constants Enforce Consistency**
- Holoflower always gold (unifies all styles)
- Sacred geometry structure identical (same SVG paths)
- Navigation layout fixed (same positions)
- Mission dots same colors across all styles (blue/green/gold/red)
- MAIA voice archetypal (not theme-dependent)

**Rule 3: Atmospheric Layers Only**
- Presets ONLY change background, particles, shadows
- Never change structural elements
- Never change interaction patterns
- Never change information architecture

**Rule 4: Professional Design Review**
- Each new preset requires designer approval
- Color accessibility tested (contrast ratios)
- Motion tested (no seizure triggers)
- Typography tested (readability)

**Rule 5: Limited Customization**
- Members can only tweak: particle density, motion speed, contrast
- No color picker (prevents garish combinations)
- No font picker (prevents readability issues)
- Minor tweaks within safe parameters
