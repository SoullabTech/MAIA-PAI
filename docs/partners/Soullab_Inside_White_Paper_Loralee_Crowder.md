## 9. Technical Architecture: What's Under the Hood

### **Multi-Tenant Infrastructure**
Each partner gets their own isolated environment:
- Dedicated database (your client data never mixes with others)
- Custom subdomain or domain (yourpractice.com)
- Shared core framework (security updates, feature improvements)
- Siloed data (HIPAA-compliant separation)

### **Soullab Inside SDK**
Developers and technical partners can access:
- `@soullab/inside` npm package
- React components with elemental design tokens
- Conversational intelligence hooks
- Pattern library from the Commons

### **Design System Tokens**
