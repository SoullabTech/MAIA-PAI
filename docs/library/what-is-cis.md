## Behind the Scenes: The Technology

CIS is built as four modular systems:

- **ConversationalEnhancer** - Removes therapy-speak, adds natural acknowledgments
- **ConversationalMagicEngine** - Real-time rhythm, back-channeling, engagement scoring
- **Elemental Refiner + Phrasebook** - Archetypal language in everyday phrases
- **ConversationFlowTracker** - Tracks conversation arc and depth

Each module can be enabled/disabled independently. They work together but don't depend on each other.

**Performance impact:** ~20-50ms latency per turn (worth it for the presence)

**Architecture:** Conservative-first (only intervenes on therapy-speak or generic responses)

**Philosophy:** Instruction-first (teach the AI how to speak), correction-second (safety net)
