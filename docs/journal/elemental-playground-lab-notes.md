- All geometry wrapped in `{showSacredGeometry && (<g opacity="0.25">...</g>)}`

- Layering order: Torus â†’ Dodecahedron â†’ Seed of Life â†’ Spirals â†’ Metatron â†’ Trigrams â†’ Polygons

### Gradient ID Fix
- Each alchemical symbol generates unique gradient ID to prevent SVG conflicts
- Format: `{element}-${Math.random().toString(36).substr(2, 9)}`
- Ensures Air and Earth horizontal lines render correctly

### Animation Philosophy
- All animations use **breathing patterns** (easeInOut)
- Staggered delays create **interference patterns**
- Different timestreams (5s - 28s) prevent static synchronization
- PathLength animations for line-drawing effects
- Scale animations centered on transformOrigin: '200px 200px'
