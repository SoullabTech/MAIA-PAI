## 1. SILENCE TEST (Heartbeat Reference)

**What it is:**
> "Run one or two full exchanges with the reflection hook disabled, note the natural conversational rhythm, then re-enable it and compare. That's your heartbeat reference for whether Maia's interjections feel organic."

**Why it matters:**
- Establishes baseline rhythm before interference
- Creates sensory control sample
- Tests whether reflections blend or clash
- Reveals if interjections are organic or forced

**How to do it:**

### Step 1: Baseline Conversation (Reflections OFF)
```bash
ELEMENTAL_REFLECTION_ENABLED=false
```

**Have 1-2 full exchanges. Notice:**
- Natural conversation rhythm
- Timing of responses
- Spaciousness between exchanges
- Where Maia naturally pauses
- Overall flow quality

**Document in one sentence:**
```
The baseline rhythm feels: _________________________________
```

### Step 2: Test Conversation (Reflections ON)
```bash
ELEMENTAL_REFLECTION_ENABLED=true
```

**Have same topic conversation. Notice:**
- Does rhythm match baseline?
- Do reflections feel organic or inserted?
- Is timing still natural?
- Does spaciousness remain?

**Compare in one sentence:**
```
With reflections, the rhythm: _________________________________
```

### Silence Test Results

**PASS:**
- Rhythm matches baseline
- Reflections feel organic
- Timing still natural
- Spaciousness preserved

**CAUTION:**
- Rhythm slightly disrupted
- Reflections mostly organic with occasional forced moments
- Timing close but occasionally off
- Spaciousness reduced but acceptable

**FAIL:**
- Rhythm significantly disrupted
- Reflections feel inserted/mechanical
- Timing feels wrong
- Spaciousness lost

**How to do it:**

### After Each Test Exchange
