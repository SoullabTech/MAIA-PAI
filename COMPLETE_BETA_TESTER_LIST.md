## Support Commands

### Check if tester is in database:
```sql
SELECT explorer_name, email, invitation_code, status
FROM explorers
WHERE email = 'their@email.com';
```

### See all testers in database:
```sql
SELECT explorer_name, email, invitation_code,
       signup_date, session_count,
       birth_chart_data IS NOT NULL as has_birth_chart
FROM explorers
ORDER BY explorer_name;
```

### Add tester to database:
```bash
# Create script based on template:
# scripts/add-[name].ts
npx dotenv -e .env.local -- npx tsx scripts/add-[name].ts
```
