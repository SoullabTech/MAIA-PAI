1. Search for these keywords (case-insensitive) across all markdown files

3. Generate elemental scores
4. Determine dominant element based on highest score

### For Manual Tagging
Users don't need to use these exact words in YAML tags. These are the words the script SEARCHES FOR in file content. Users can tag broadly (e.g., `tags: [fire, transformation]`) or specifically (e.g., `tags: [breakthrough, catalyst, turning-point]`).
