## How to Use

### Quick Start

1. **Use Dune component classes:**
```tsx
// Buttons
<button className="btn-spice">ENTER THE DESERT</button>
<button className="btn-fremen">View Sietch</button>

// Cards
<div className="card-sietch">Your content</div>

// Inputs
<input className="input-water" placeholder="Enter your vision..." />
```

2. **Use Dune colors:**
```tsx
<div className="bg-spice-orange text-sand-white">
  Spice orange background
</div>
```

3. **Use Dune typography:**
```tsx
<h1 className="font-cormorant text-dune-hero text-spice-gradient">
  The Litany Against Ignorance
</h1>
<p className="font-cinzel text-dune-body text-deep-sand">
  Body text in Cinzel
</p>
```

4. **Use Dune animations:**
```tsx
<div className="animate-spice-pulse">Pulsing like spice</div>
<div className="animate-sandworm-spiral">Loading...</div>
```
